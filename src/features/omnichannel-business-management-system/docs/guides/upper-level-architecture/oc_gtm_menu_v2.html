💡 v2.0 菜单设计的三大核心洞察
1. 每个角色都有"独占主阵地"
BI → 市场洞察 / PG → 产品中心 + GTM 项目 / RG → 价格中心 / DP → PSI 中心。系统登录后可直接跳转主阵地。
2. 菜单边界 = GTM 职责边界
"GTM 制定下限、定策略、看全局"，不做具体执行。广告投放/Listing/订单等执行菜单全部剔除。
3. 结构可直接用于 RBAC 设计
