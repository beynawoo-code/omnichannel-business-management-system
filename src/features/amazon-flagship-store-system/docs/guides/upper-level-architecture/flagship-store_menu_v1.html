💡 v1.0 菜单设计的四大核心洞察
1. 每角色都有"独占主阵地"
CP → Tag 中心 + 联盟中心 / SO → 站点与场景 / DA → 旗舰店看板 / BO → 目标管理 + 复盘报告。登录后直接跳主阵地。
2. 7 大看板用 Tab 方式组合
① 模块内 7 Tab 满足"既要聚合看，又要单独看"；统一筛选器与同比/环比贯穿所有 Tab，支持统一分析口径。
3. 强约束写入菜单层
6 项 K1–K6 从设计阶段就确保 Tag 规范、映射快照、指标重算、审计等系统强制，彻底告别 Excel + DeepSeek 手工流。
4. 结构可直接用于 RBAC
⭐/● = 读写，○ = 只读，— = 不可见。四色角色矩阵可直接映射为权限配置表，驱动 RBAC 实现。
